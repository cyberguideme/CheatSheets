## File Operations

* read a file line by line into a list

	* If you want the \n included:

	```python
		with open(fname) as f:
	    	content = f.readlines()
	```

	* If you do not want \n included:

	```python
	with open(fname) as f:
	    content = f.read().splitlines()
	```
